# Third-Party Tools

This directory contains debugging and analysis tools for various third-party applications and services.

## Available Tools

### `claude-debug/`
Node.js debugging utilities for diagnosing stuck Claude Code sessions.

## Future Tools

This directory structure can accommodate debugging tools for:
- Browser automation tools (Playwright, Selenium)
- Database tools (Redis, PostgreSQL)
- API testing tools (Postman, Insomnia)
- Development servers (Next.js, Vite)
- And other third-party tools that may need debugging

Each tool set gets its own subdirectory with appropriate documentation.
