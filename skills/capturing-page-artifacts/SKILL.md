# Capturing Page Artifacts

Use this skill when the output is visual: screenshots, PDFs, or screenshot-backed test evidence.

## Quick start

- `screenshot` captures PNG artifacts
- `pdf` exports print-style PDFs
- HAR capture workflows can embed screenshots and DOM snapshots as annotations

## Use this skill for

- choosing filenames and output directories
- sequencing screenshots across a workflow
- capturing mobile, tablet, dark-mode, or full-page variants
- combining visual artifacts with HAR tagging
